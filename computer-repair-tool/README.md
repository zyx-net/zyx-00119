# 电脑维修接单系统

本地桌面工具，纯前端实现，无需任何后端或数据库。

---

## 启动方式

### 方式一：直接双击（最简单）
双击 `index.html` 即可在默认浏览器中打开使用。

### 方式二：本地 HTTP 服务器（推荐，避免某些浏览器的本地文件限制）
```bash
# 在项目目录下执行
python -m http.server 8080
# 然后访问 http://localhost:8080
```
或
```bash
npx serve .
```

## 数据存储说明

所有数据保存在浏览器 `localStorage` 中，key 前缀为 `crp_`：
- `crp_orders` - 工单
- `crp_parts` - 配件配置
- `crp_labor` - 工时配置
- `crp_quotes` - 报价版本（含快照价格，永不修改）
- `crp_history` - 操作历史
- `crp_rollback_records` - 撤回记录
- `crp_termination_records` - 终止记录

**关键保证**：报价一旦生成，版本内的配件价/工时费是快照，后续修改配置不影响历史版本。
